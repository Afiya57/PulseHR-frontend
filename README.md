# Employee Management System - Frontend

A clean and minimal React frontend for the Employee Management System.

## Features

- **Authentication**
  - User login and registration
  - JWT-based authentication
  
- **Dashboard**
  - Overview statistics (Admin only)
  - Employee metrics and department breakdown
  
- **Employee Management**
  - View all employees with search and filter
  - Add/Edit/Delete employees (Admin only)
  
- **Attendance Tracking**
  - Mark attendance (Admin)
  - View attendance history
  
- **Leave Management**
  - Apply for leave
  - Approve/Reject leave requests (Admin)
  
- **User Profile**
  - View and edit personal information
  - Change password

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 19
- Vite
- CSS3 (No frameworks - clean, custom styles)
